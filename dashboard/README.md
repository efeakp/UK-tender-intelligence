# Nordic Energy Tender Dashboard

React + Vite frontend for the Nordic Energy Tender Intelligence platform.

See the [root README](../README.md) for full project documentation, quick start instructions, and API reference.

## Development

```bash
npm install
npm run dev       # http://localhost:5173
```

Connects to the FastAPI backend at `http://localhost:8000` by default.  
Change `API_BASE` at the top of `src/App.jsx` to point at a different environment.

## Build

```bash
npm run build     # outputs to dist/
```
